Database Connection
There is no point of using php if you dont need to connect your website to a database.
We are going to be using a MySQL database in all our practises.Note that MySQL database can be manipulated using DOM or Mysqli.
We are going to use the latter(Mysqli).

Creating a database
1.open your browser
2.type in "localhost/" in your addressbar
3.click phpMyAdmin
4.Create a database
5.Create a table

NB:Make sure you put a primary key on your table.

Establishing a Connection
create a varible to hold your database connection,with the format :
$db=mysqli_connect('localhost','database','password','database name');

  • $db is a variable allocated to our connection,it can have any name but l just preffer using $db,others preffer using $conn

  • mysqli_connect is a predefined function for database connections

  • localhost it is the servername

  • username it is the username(in our case root)

  • password it is the password of the main database

  • database name is the name of the database which you create in phpMyAdmin


code example
<?php
session_start();
$db=mysqli_connect('localhost','root','','users');
session_destroy();

?>
Note that on the above illustration l put empty qoutes('') for our password field because we didnt put a password to our root database.Localhost is our server,root is our database,users is our database name.

Adding items to the database

Adding items to the database is one of the most significant operations on the database.As l mentioned earlier we are going to be using Mysqli for our database operations

Here we are going to use our database with database name users and a table name credentials
$sql="INSERT INTO credentials (username,password) values('$username','$password')";
mysqli_query($db,$sql);
  • $sql is a variable allocated to our inserting string,NB:it can have any name

  • INSERT INTO is an SQL statement for inserting items into a table

  • credentials is our table name we intend to insert items into

  • (username,password):our table column names

  • values('$username','$password'):putting values(data) from the user input to the table columns.

  • mysqli_query():is a predefined function for executing the sql statement with two arguments,$db(connection string) and $sql(sql statement)



code example

session_start();
$db=mysqli_connect('localhost','root','','users');

1.mysql_fetch_assoc()
Here we have introduced one more predefined function,mysql_fetch_assoc(),it is used to fetch or take database values and make them readable through html.it has only one argument,which is suppossed to be a mysqli_query() which we inially assigned to $result,that is $result=mysqli_query($db,$sql);.

2.$rows=mysql_fetch_assoc($result)
Now we have a new variable we declared,$rows.It is used to hold our mysql_fetch_assoc() function.We then used it to directly fetch and display specific database values into specific html table cells($rows['username']).

3.Note that there is some php code we wrote inside the html code,the while statements starts inside a first php code of the table and ends in the second php code.
